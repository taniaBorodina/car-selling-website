# 🚗 Car Selling Website

##  Project Description

The goal of this project is to create a web application that allows users to browse available cars and simulate the process of purchasing them online.

The application includes a product listing, shopping cart functionality, checkout form, and order confirmation page.

---

##  Live Demo

##  Pages

* index.html – homepage with car listings
* checkout.html – checkout form
* success.html – order confirmation page

---

##  Technologies

* HTML
* CSS
* JavaScript
* LocalStorage (for data persistence)

---

##  Application Logic

The application uses JavaScript to dynamically manipulate the DOM. Users can add and remove products from the cart, update quantities, and see the total price updated in real time.

Cart data and user input are stored in LocalStorage, which allows data to persist after page refresh.

---

##  Data Structure (LocalStorage)

CartItems:

* title (string)
* price (string)
* quantity (number)
* carImg (string)

FormData:

* name (string)
* address (string)
* deliveryDate (string)
* paymentMethod (string)
